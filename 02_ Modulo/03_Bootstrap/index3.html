Como colocar o Bootstrap localmente

  1. Ir no link e baixar: Bootsrap, compiled CSS and JS

  2. Salvar o arquivo zipado na pasta trabalhada
  3. Extrair
  4. Chamar os links
    1º bootstrap.css: 
    < link rel="stylesheet" href="bootstrap-5.3.0-dist/css/bootstrap.css" >

    2º bootstrap.js: 
    < script src="bootstrap-5.3.0-dist/js/bootstrap.js" >
  5. E sua configuração está feita!
